# BACKEND_INVENTARIO
BACKEND EN NODE.JS PARA SISTEMA DE INVENTARIO

Se requiere archivo .env dentro de la ruta /BackEnd-PIC
con los siguientes parametros

- HOST=
- PORT=
- DB_USER=
- DB_PASS=
- DATABSE=
- MONGO_LOCAL=mongodb://localhost/Inventario
- DEVELOPMENT=true
- APP_SECRET=
- JWT_SECRET=


Comandos 
 - npm start: Iniciar aplicacion
